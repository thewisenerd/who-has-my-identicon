who-has-my-identicon
====================


_you're ~~identicon is~~ not special. there are a lot of others ~~identicons~~
just like you._

howto
-----

1. generate hashes (a 1.2 gigabyte file with hashes of numbers from 1 to 30mil).

```
./hashgen.py > hashes
```

2. install requests
```
pip install requests
```
